1. Perbedaan Selector ID dan Class

ID digunakan untuk satu elemen unik (menggunakan #), sedangkan class digunakan untuk banyak elemen (menggunakan .).

2. Mengubah Warna Teks

Gunakan properti color. Contoh: p { color: blue; } akan membuat teks paragraf berwarna biru.

3. Pentingnya CSS dalam Web

CSS membantu membuat tampilan web lebih menarik, rapi, dan mudah digunakan, serta memisahkan desain dari struktur HTML.

4. Sintaks Dasar CSS

Sintaks CSS: selector { property: value; }. Contoh: h1 { font-size: 24px; }

5. Apa Itu Cascading?

Cascading berarti urutan gaya yang berlaku. Style terakhir yang valid akan menimpa yang sebelumnya jika selector-nya sama.

6. Mengatur Tata Letak

Gunakan display: flex;, grid, atau position untuk mengatur tata letak elemen HTML.

7. Desain Responsif

Desain responsif menyesuaikan tampilan dengan berbagai ukuran layar (HP, tablet, PC), agar user experience tetap optimal.

8. Gaya dengan ID dan Class

Contoh ID: #judul { color: red; }
Contoh Class: .menu { font-weight: bold; }

9. Pseudo-Class

Pseudo-class digunakan untuk kondisi tertentu, contoh: a:hover { color: green; } saat link disentuh kursor.

10. Evaluasi & Peningkatan CSS

Lihat kecepatan loading, keterbacaan, dan kenyamanan. Memperbaiki dengan memperjelas struktur, warna, dan tata letak.
